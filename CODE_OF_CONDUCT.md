# Code of Conduct

## Наши принципы

- Уважительное общение
- Конструктивная обратная связь
- Фокус на решении задач

## Недопустимо

- Оскорбления и токсичное поведение
- Дискриминация в любой форме
- Публикация чужих секретов и чувствительных данных

## Разрешение конфликтов

Спорные вопросы решаются через факты, воспроизводимость и прозрачные технические аргументы.
