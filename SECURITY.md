# Security Policy

## Поддерживаемые версии

Поддерживается актуальная версия из ветки `master`.

## Как сообщить об уязвимости

Если вы нашли уязвимость:

1. Не публикуйте детали публично сразу.
2. Подготовьте краткое описание:
- что уязвимо
- как воспроизвести
- потенциальное влияние
3. Передайте информацию владельцу проекта приватно.

## Рекомендации по безопасной эксплуатации

- Храните `FLASK_SECRET_KEY` и `DATABASE_URL` только в переменных окружения.
- Не сохраняйте OCR API токены в коде/репозитории.
- Используйте HTTPS в продакшене.
